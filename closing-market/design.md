# 클로징마켓 앱 인터페이스 디자인

## Screen List

1. **홈 (Home)** - 메인 화면, 카테고리/추천 물품/추천 업체/최신 물품
2. **카테고리 (Category)** - 전체 카테고리 그리드 뷰
3. **등록하기 (Register)** - 상품 등록 폼
4. **채팅 (Chat)** - 채팅 목록
5. **마이페이지 (MyPage)** - 프로필 및 설정
6. **상품 상세 (Product Detail)** - 상품 이미지/정보/액션
7. **업체 상세 (Business Detail)** - 업체 소개/포트폴리오/리뷰
8. **검색 (Search)** - 키워드/필터 검색
9. **알림 (Notifications)** - 알림 목록

## Primary Content and Functionality

### 홈 화면
- 상단: 앱 이름 텍스트 "클로징마켓" + 검색/알림 아이콘
- 카테고리: 가로 스크롤, 아이콘+텍스트 (카페, PC방, 식당, 헬스장, 사무실, 창고재고, 사업양도)
- 추천 사업정리 물품: 카드 리스트 또는 빈 상태 메시지
- 추천 업체: 카드 리스트 (아이콘 + 업체명 + 설명 + 문의 버튼)
- 최신 등록 물품: 카드 리스트 또는 빈 상태 메시지

### 카테고리 화면
- 2열 그리드 형태의 카테고리 카드
- 각 카테고리 탭 시 해당 카테고리 상품 목록으로 이동

### 등록하기 화면
- 이미지 업로드 영역 (대표 1장 + 추가 최대 30장)
- 제목, 설명, 가격, 수량, 거래 지역, 업종, 거래 유형, 판매 상태 입력 폼

### 채팅 화면
- 채팅방 목록 (상대방 프로필, 마지막 메시지, 시간, 읽지 않은 수)
- 빈 상태: "진행 중인 채팅이 없습니다."

### 마이페이지
- 프로필 영역 (사진, 닉네임, 인증 배지)
- 메뉴 리스트: 내 상품, 구매내역, 찜 목록, 채팅 목록, 최근 본 상품, 거래 후기, 알림 설정, 계정 설정

### 상품 상세
- 이미지 슬라이더
- 가격, 제목, 위치, 판매자 정보
- 조회수, 찜 수, 등록일
- 하단 고정: 채팅하기, 전화하기, 찜하기 버튼

### 업체 상세
- 업체 대표 이미지
- 업체 소개, 위치(지도), 포트폴리오, 리뷰
- 하단: 문의하기, 전화하기 버튼

## Key User Flows

1. **상품 탐색 플로우**: 홈 → 카테고리 선택 → 상품 목록 → 상품 상세 → 채팅하기
2. **업체 문의 플로우**: 홈 → 추천 업체 → 업체 상세 → 문의하기/전화하기
3. **상품 등록 플로우**: 탭바 '등록하기' → 이미지 업로드 → 정보 입력 → 등록 완료
4. **검색 플로우**: 검색 아이콘 → 키워드 입력 → 필터 적용 → 결과 목록 → 상품 상세

## Layout Principles

- 모바일 세로 모드(9:16) 기준 설계
- 한 손 조작 최적화 (하단 탭바, 하단 고정 CTA)
- 카드 간 간격: 12px
- 화면 좌우 패딩: 16px
- 카드 border-radius: 12px
- 섹션 타이틀: 18px bold, 좌측 정렬
